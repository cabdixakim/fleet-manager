# Architecture Overview

## Tech Stack
- Frontend: React + Vite + TypeScript + Tailwind
- Backend: Express + TypeScript
- Database: PostgreSQL + Drizzle ORM

## Monorepo Structure
- `artifacts/` - App artifacts (api-server, web)
- `lib/` - Shared libraries (db, api-zod, api-client-react, etc)
- `scripts/` - Utility scripts

## Key Modules
- Dashboard, Batches, Trips, Trucks, Drivers, Clients, Subcontractors, Invoices, Payroll, Expenses, Audit Log, Login, Setup

## Planned Improvements
- Modular backend (services, validators, policies)
- Enhanced UI/UX
- Role-based access control
- Document generation
- Reporting
